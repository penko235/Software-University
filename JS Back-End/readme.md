Theaters 📽️
This is an application for creating and editing plays in theaters.

The DB used is local (Robo 3T)

Installation 🔧

Install node_modules
- express
- express-handlebars
- cookie-parser 
- express-validator 
- mongodb
- mongoose 
- bcrypt 
- jsonwebtoken
- nodemon


Start 🚀
Start the project by typing in the terminal

Open with localhost/8888
npm run start
